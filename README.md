# -Explorateur-de-Pays
mon troisième projet

Explorateur de Pays est une application interactive qui permet aux utilisateurs de découvrir facilement des informations essentielles sur n’importe quel pays du monde.
Le projet offre une interface simple et intuitive pour rechercher un pays, visualiser ses données principales et mieux comprendre sa culture, sa géographie et son économie.

Objectif du projet

Rendre l’exploration géographique simple, éducative et accessible à tous.
Ce projet peut servir d’outil pédagogique, de support d’apprentissage ou même de base pour une application plus complète (voyage, culture générale, statistiques…).
